## 第一部分：项目概述

### 项目名称
**QuantumTrader AI**

### 项目定位
全自动AI量化交易系统，科幻风格UI，面向开发者和量化爱好者的开源交易平台。

### 核心功能
- **多交易所支持** - 目前支持欧易（OKX），架构可扩展
- **AI策略系统** - 6种经典策略 + AI分析 + 策略回测 + 实盘运行
- **三层风控系统** - 系统级+策略级+单交易级，三级熔断机制
- **实盘+模拟盘** - 模拟盘优先，支持切换到实盘
- **实时数据推送** - WebSocket架构，行情/订单/持仓实时更新
- **期权交易** - T型报价、希腊字母、完整交互流程
- **本地AI模型** - 支持llama.cpp本地推理 + 云端DeepSeek混合调度

### 技术栈

**后端技术栈：**
- Python 3.12+
- FastAPI 高性能异步框架
- SQLAlchemy ORM
- SQLite 数据库（可扩展PostgreSQL）
- Redis（预留，目前未使用）
- JWT Token认证
- WebSocket 实时通信
- AES-256 加密（Fernet）

**前端技术栈：**
- Vue 3 + TypeScript
- Vite 构建工具
- Pinia 状态管理
- ECharts 图表库
- Element Plus UI组件库
- Vue Router 路由
- TailwindCSS 样式框架

**实时通信：**
- WebSocket
- 订阅/取消订阅机制
- 心跳保活
- 自动重连（指数退避算法）

**AI能力：**
- DeepSeek API（云端）
- 本地llama.cpp（开发中）
- GGUF模型格式
- 混合调度架构

**通知服务：**
- PushPlus 微信推送

**数据库：**
- SQLite + SQLAlchemy ORM
- 支持迁移到PostgreSQL

### 项目当前完成度
**约 82%**

### 开源协议
**MIT License**

### 项目特点
1. **深色科幻霓虹风格** - 专业交易终端视觉体验
2. **中英文双语架构** - 支持国际化
3. **模块化设计** - 前后端分离，易于扩展
4. **API密钥加密存储** - AES-256加密，脱敏显示
5. **风控优先** - 三层风控体系，资金安全第一
6. **真实数据** - 对接真实交易所API，非模拟假数据
7. **开源友好** - 代码规范，注释完整，易于二次开发
