# QuantumTrader AI 开发路线图

## 🎯 项目愿景

打造一个开源的、AI驱动的量化交易系统，让每个人都能轻松构建和运行自己的交易策略。

---

## ✅ 已完成 (v0.1.0)

### 后端基础
- [x] FastAPI 后端框架
- [x] SQLite + SQLAlchemy ORM
- [x] JWT 用户认证系统
- [x] API 密钥加密存储 (AES-256)
- [x] 统一 API 响应格式
- [x] 全局异常处理

### 交易功能
- [x] 欧易 (OKX) 模拟盘对接
- [x] 账户余额查询
- [x] 持仓查询
- [x] 限价单/市价单下单
- [x] 撤单功能
- [x] 订单历史查询
- [x] 成交记录查询

### 行情数据
- [x] 实时行情数据

- [x] 买卖盘深度数据
- [x] 交易对搜索

### 策略系统
- [x] 策略回测框架
- [x] 策略实盘运行引擎
- [x] 6个经典策略：

  - [x] RSI 超买超卖
  - [x] MACD 金叉死叉
  - [x] 布林带 (Bollinger Bands)
  - [x] 定投策略 (DCA)
- [x] 策略运行状态持久化
- [x] 策略运行日志

### 风控系统

- [x] 三级熔断机制 (5%/10%/15%)
